# portfolio-hilaire
Evolutions des modèles économiques des Câbles sous-marins 

Les Câbles sous-marins constituent aujourd'hui, des infrastructures stratégiques devenues le système nerveux de l’économie mondiale. Cette production vise à apporter une tentative d’explication de leur fonctionnement, tant sur le plan du modèle économique que sur le plan du fonctionnement pratique de ces câbles. Ainsi, après avoir abordé l’histoire des câbles, leurs origines et leur fonctionnement, seront étudiés les principaux acteurs du milieu, ainsi que les enjeux et les potentielles évolutions.
